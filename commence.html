Une technologie éprouvée qui n'a pas d'équivalent en terme de souplesse et de simplicité

Si vous voulez une solution simple, pourtant parfaitement adaptée à votre métier, une application fiable qui puisse évoluer facilement et à moindres coûts, une application dont vous puissiez garder le contrôle, ne cherchez pas plus loin : c’est Commence qu’il vous faut.

Depuis 20 ans, nous utilisons Commence pour satisfaire, rapidement et à moindre coût, les besoins de centaines d’entreprises dans les secteurs d’activité les plus divers et pour des applications toujours plus spécifiques. Nous n’avons jamais trouvé de technologie équivalente.

L’interface de Commence permet de modifier, à la volée, tous les éléments de votre application dans la limite des droits qui vous ont été accordés : tables, champs, vues, filtres, requêtes, masques de saisie, états, agents… Avec Commence, il peut nous suffire de quelques heures pour créer une toute nouvelle application. Bien sûr, il faudra beaucoup plus longtemps pour réaliser un projet complet de CRM. Mais, et c’est encore, un des avantages de ce logiciel de développement rapide, sa rapidité de paramétrage facilite une conduite de projet itérative qui garantit des résultats rapides et une adoption accrue par les utilisateurs (voir notre démarche de gestion de projet).

Commence inclut :
  • Une base de données ouverte entièrement modifiable
  • Un moteur de Workflow
  • Un moteur de synchronisation
  • Une API (interface de programmation)

Malheureusement, Commence a aussi ses limites. Dans la mesure où la base de données fonctionne sur un mode de synchronisation, cette technologies est inadaptée pour des applications incluant un grand nombre d’utilisateurs (100+) et de grands volumes de données (500.000 +). Bien qu’il soit possible d’y accéder via un navigateur (via des outils tiers type Go6Global de GraphOn), ce n’est pas non plus une technologie FullWeb. En conséquence, Commence ne sera pas notre outil de choix dans le cas d’une application qui demande une interaction importante avec le Web (site Web, portail end-user, …).
